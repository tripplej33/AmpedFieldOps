# Diagnostic Questions (Prompt for More Context)

Use these when requirements are unclear:

- What exact outcome do you want from this change?
- Which environment are you targeting (local, staging, production)?
- What domain or IP will the frontend be served from?
- Should we use Supabase Cloud or local Supabase?
- Any constraints on ports, SSL, or networking?
- Do you prefer a specific branch naming convention?
